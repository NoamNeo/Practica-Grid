# Práctica Grid - Moodboard Personal

## Autor/a
Gabriela Loira Lorenzo

## Descripción
Práctica de desarrollo web que consiste en un moodboard/portfolio personal. El proyecto muestra los intereses, filosofía y proyectos personales del autor mediante un diseño basado en cuadrícula (CSS Grid) con temática de cafetería.

La página incluye:
- Información personal y filosofía
- Lista de intereses (juegos de cartas, videojuegos, series, dibujo, escritura)
- Imágenes decorativas
- Sección de proyectos personales con enlaces a GitHub
- Diseño responsive adaptable a móviles y tablets

## Tecnologías utilizadas
- **HTML5** - Estructura semántica del documento
- **CSS3** - Estilos y diseño visual
- **CSS Grid** - Layout principal de la página
- **CSS Flexbox** - Alineación de elementos
- **CSS Custom Properties** - Variables para el tema de colores
- **Google Fonts** - Tipografía (IBM Plex Mono)
- **CSS Media Queries** - Diseño responsive
